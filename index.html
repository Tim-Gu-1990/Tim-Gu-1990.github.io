node.js的作用

  1. node 采用 EcmaScript 进行编码，没有 bom 和 dom ，和浏览器不同

  2. node.js 可以读写文件

    fs 是 file-system 的缩写，在 node 中如果想要进行文件操作，就必须引入 fs 这个核心模块

    在 fs 中提供了所有文件操作相关的 API；例如：fs.readFile 就是用来读取文件的

  3. 使用 Node 构建一个 Web 服务器
